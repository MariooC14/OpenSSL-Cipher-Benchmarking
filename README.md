# Notes for the reader
This source code was developed using CLion. The CMakeLists.txt file is included and shows that it can generate two executables.
- `main.cpp` contains the code for part 1, benchmarking AES, ARIA and Camellia, for a total of 72 configurations
- `triple-des-benchmark.cpp` contains the code for part 2, benchmarking Triple DES, for a total of 8 configurations
- `openssl-benchmarks.cpp` contains the utility functions used by both part 1 and 2. This file builds up on the code provided in the lab instructions.
